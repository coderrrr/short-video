# C端用户使用手册

## 欢迎使用企业内部短视频平台

企业内部短视频平台是一个专为员工打造的知识分享和文化交流平台。您可以在这里：

- 📹 分享工作知识和经验
- 🌟 记录美好生活瞬间
- 📚 学习同事分享的内容
- 💬 与同事互动交流
- 🎯 制定个性化学习计划

## 快速开始

### 1. 访问平台

在企业App中找到"短视频平台"入口，点击进入。

### 2. 首次登录

使用您的企业员工ID和密码登录。登录后系统会根据您的岗位和部门推荐相关内容。

## 功能指南

### 一、浏览内容

#### 1.1 推荐信息流

**位置**：首页

**功能**：
- 系统根据您的岗位、兴趣和观看历史智能推荐内容
- 下拉刷新获取最新推荐
- 上拉加载更多内容

**操作**：
- 点击视频卡片进入播放页面
- 左右滑动切换视频
- 双击视频点赞

#### 1.2 关注信息流

**位置**：首页 - 关注标签

**功能**：
- 查看您关注的创作者发布的最新内容
- 按时间倒序排列

**操作**：
- 点击创作者头像查看其个人主页
- 点击视频观看内容

#### 1.3 分类浏览

**位置**：首页 - 分类标签

**功能**：
- 按内容类型浏览（工作知识、生活分享、企业文化等）
- 支持多级分类导航

**操作**：
1. 选择感兴趣的分类
2. 浏览该分类下的内容
3. 点击子分类进一步筛选

### 二、搜索和筛选

#### 2.1 搜索内容

**位置**：首页 - 搜索图标

**功能**：
- 搜索视频标题、描述、标签
- 查看搜索历史
- 获取搜索建议

**操作**：
1. 点击搜索框
2. 输入关键词（如"Python"、"团建"）
3. 点击搜索或选择搜索建议
4. 查看搜索结果

**提示**：
- 支持多个关键词搜索
- 搜索历史会自动保存
- 可以清除搜索历史

#### 2.2 多维筛选

**位置**：搜索结果页 - 筛选按钮

**功能**：
- 按内容类型筛选
- 按标签筛选
- 按发布时间筛选

**操作**：
1. 点击筛选按钮
2. 选择筛选条件
3. 点击确定查看筛选结果
4. 点击清除筛选恢复默认

### 三、视频播放

#### 3.1 播放控制

**功能**：
- 播放/暂停
- 进度调节
- 音量控制
- 倍速播放（0.5x - 2x）

**操作**：
- 点击屏幕：播放/暂停
- 拖动进度条：跳转到指定位置
- 点击倍速按钮：选择播放速度
- 上下滑动：调节音量
- 左右滑动：切换视频

#### 3.2 视频信息

**显示内容**：
- 视频标题和描述
- 创作者信息
- 发布时间
- 观看数、点赞数、评论数
- 标签

**操作**：
- 点击创作者头像：查看创作者主页
- 点击标签：查看相关内容

### 四、互动功能

#### 4.1 点赞

**操作**：
- 双击视频或点击点赞按钮
- 再次点击取消点赞

**说明**：
- 点赞会影响推荐算法
- 可以在"我的点赞"中查看所有点赞的视频

#### 4.2 收藏

**操作**：
- 点击收藏按钮
- 再次点击取消收藏

**说明**：
- 收藏的视频可以在"我的收藏"中查看
- 收藏不会通知创作者

#### 4.3 标记和笔记

**操作**：
1. 点击标记按钮
2. 输入笔记内容
3. 点击保存

**说明**：
- 笔记仅自己可见
- 可以在"我的标记"中查看和编辑笔记

#### 4.4 评论

**操作**：
1. 点击评论按钮
2. 输入评论内容
3. 使用@提及其他用户
4. 点击发送

**说明**：
- 评论会通知创作者
- @提及会通知被提及的用户
- 可以回复其他人的评论

#### 4.5 分享

**操作**：
1. 点击分享按钮
2. 选择分享方式（企业微信）
3. 选择接收人
4. 发送

**说明**：
- 分享会增加视频的分享计数
- 接收人点击链接可直接观看视频

#### 4.6 关注

**操作**：
- 在创作者主页点击关注按钮
- 再次点击取消关注

**说明**：
- 关注后可以在"关注信息流"中看到该创作者的新内容
- 可以在"我的关注"中管理关注列表

### 五、创作内容

#### 5.1 上传视频

**操作**：
1. 点击首页"+"按钮
2. 选择拍摄或从相册选择
3. 选择视频文件

**要求**：
- 支持格式：MP4、MOV、AVI
- 文件大小：最大500MB
- 时长：建议1-10分钟

#### 5.2 编辑视频

**功能**：
- 视频裁剪
- 音量调节
- 封面选择

**操作**：
1. 上传视频后进入编辑页面
2. 拖动时间轴选择裁剪范围
3. 调节音量滑块
4. 选择视频帧或上传图片作为封面
5. 点击下一步

#### 5.3 添加元数据

**必填项**：
- 标题（最多200字符）

**选填项**：
- 描述（最多500字符）
- 内容类型（工作知识、生活分享、企业文化等）
- 自定义标签

**操作**：
1. 填写标题和描述
2. 选择内容类型
3. 添加标签（用逗号分隔）
4. 点击保存为草稿或提交审核

#### 5.4 草稿管理

**位置**：个人中心 - 我的发布 - 草稿

**功能**：
- 查看所有草稿
- 继续编辑草稿
- 删除草稿
- 提交审核

**操作**：
1. 进入草稿列表
2. 点击草稿进入编辑
3. 完成编辑后提交审核

#### 5.5 审核流程

**状态说明**：
- 草稿：未提交审核
- 审核中：已提交，等待审核
- 已发布：审核通过，已发布
- 已驳回：审核未通过，可修改后重新提交

**注意事项**：
- 审核中的内容无法编辑
- 审核结果会通过通知告知
- 被驳回的内容可以查看驳回原因

### 十、常见问题

#### Q1: 为什么我的视频一直在审核中？

A: 视频审核通常需要1-2个工作日。如果超过3个工作日仍未审核，请联系管理员。

#### Q2: 如何提高视频的曝光率？

A: 
- 选择合适的标题和描述
- 添加相关标签
- 选择正确的内容类型
- 制作高质量的内容
- 在合适的时间发布

#### Q3: 可以删除已发布的视频吗？

A: 已发布的视频无法自行删除，如需删除请联系管理员。

#### Q4: 为什么我的视频被驳回？

A: 查看驳回原因，常见原因包括：
- 内容不符合企业文化标准
- 视频质量不佳
- 标题或描述不当
- 违反内容规范

#### Q5: 如何成为KOL？

A: KOL由管理员指定。如果您在某个领域有专业知识并愿意持续分享，可以向管理员申请。

#### Q6: 视频播放卡顿怎么办？

A: 
- 检查网络连接
- 降低视频质量设置
- 清除应用缓存
- 重启应用

#### Q7: 如何举报不当内容？

A: 
1. 在视频详情页点击"..."按钮
2. 选择"举报"
3. 选择举报原因
4. 提交举报

#### Q8: 忘记密码怎么办？

A: 使用企业统一认证系统重置密码。

❌ 违反法律法规的内容

❌ 虚假信息和谣言
